# Owais Ali Khadim Batti - Portfolio

Personal portfolio for Owais Ali Khadim Batti, a Computer Science student focused on AI, machine learning and emerging technologies.

## Featured work

- OpenClaw - multi-agent orchestration on Raspberry Pi
- Browzie - context-aware AI webpage assistant
- MedDoc - guarded medical report analysis with OCR and LLMs
- Cube Collision - published physics-based mobile game

## Stack

React 19, Vite 8, Tailwind CSS 4, hand-written WebGL, React Hook Form and EmailJS/mailto fallback.

## Local development

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run preview
```
